## 没有退出图标

刚装好 Gnome 进部分软件，右上角的最大化最小化退出按钮不会显示……但你点那个地方还是会退出，非常不爽就是了，我也不知道是啥原因。

但是装了个主题就解决了，我用的 White-Sur 主题，感觉挺好看的，好吧其实也是不知道用啥主题。之前用过这个，感觉还不赖，就一直用了，之后如果用腻了就换一个试试。

## 基础扩展

大多数安装 archlinux 的教程都是装 kde 是有道理的，会方便很多。不过 Gnome 和 kde 都有不错的地方。kde 我好像没装过扩展，也不知道能不能装，系统设置里能搞定很多事情，而 Gnome 基本啥都是装扩展，不过它扩展多。

![](https://image.wuyi.host/20251117102940498.png)

这是我的桌面目前的样子，感觉看起来挺舒服的。默认的其实也还行，开几个必要插件就能用了。

![](https://image.wuyi.host/20251117104137210.png)

我自己装的是这几个，AppIndicator 那个是显示系统托盘的，就是 QQ 那些小图标。Dash to Dock 以前倒是用过，很多人都会推荐的，就是我侧边的栏。只装这两个差不多也可以了。

Dash to Panel 的功能基本和 Dash to Dock 一样的，其实二选一就行，但是我这里用 Dash to Panel 只当顶部那个栏了，感觉很好看 hhh。而且原本换背景主题啥的，可能导致顶部字体显示不清楚，这个插件只需要稍微调下颜色渐变透明度啥的，就可以又好看又清晰。以及默认的那个活动按钮也可以通过这个隐藏，我感觉那个真没啥用吧，还不好看。不过很多插件都可以隐藏它。

Desktop 那个插件，可以让桌面显示图标，Gnome 默认会禁用桌面图标，非常的不方便，Ubuntu 会自带类似的插件。

Input Method 那个，我只知道它可以让输入法 UI 变好看一些，但是不知道还有啥其他的作用。
