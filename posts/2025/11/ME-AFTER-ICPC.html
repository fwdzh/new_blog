Me After ICPC ​

上上周末，武汉区域赛，也许是我本科期间的最后一场 ICPC 比赛，开始了两年的故事，也该完结了。

以前看过许多退役小作文，不过好像很少看到 bad ending 的吧，基本都是拿了很多奖的人。我写的这篇，与他们那些人的不一样吧，我把知乎账号也注销了。或许可以不用写的，但是这两年，感觉挺不赖的，写写试试。

昨天的南京区域赛，我的好朋友 GUAIKATTO 参加了，可惜差一点。他比我努力得多，并且只学了一年，幸运的是，他还会打一年，加油，下次一定会赢的！

我自己的话，最后一场也打得很差，我打过两次区域赛，2024 年的昆明和 2025 年的武汉。也许我不算是打过 xcpc，我从未在区域赛拿过奖。大概就是，我现在也不知道我排名多少来着，并且我永远不会去翻那一场的榜单了。

虽然是早已经猜到的结局了，非常开心还有好朋友在我身边陪我一起度过最后一次比赛。本以为我是 bad ending 的，但是当我想到这两年，遇到了许多朋友与老师，就觉得自己非常幸福，因此即便是打铁，也不能算是坏结局吧。我觉得，所有人都对我很好，谢谢你们！

第三章 C 语言 ​

大一军训后才开始上课，是在国庆之后了，很感谢我们的景老师，我们班学 C 语言的书是我们老师自己选的，和其他班的书都不一样。这本书有非常多的习题，有配套的 OJ，让当时的我学习基础语法啥的还算顺利。

中间一直是连着交的，哈哈哈，这个有点搞笑哈，全是答案错误，其实前面的页也差不多。凡是我交代码，我习惯了一直答案错误编译错误之类的，偶尔也有点被这打击到，就是很难自己一次 AC。有些题目也经常自己找不出来哪里有错误，那时候 ai 不像现在这么好用吧，那玩意也很难帮忙找到代码的问题。我一般都是尽量自己看，但如果看了很久很久，都没有发现问题，我就会找老师求助。

我记得有一次，有问题问老师，但我又有点不好意思。就给老师发了个截图，说我室友有个题目不会写，然后老师问是哪个专业的，说都到数组了，挺超前的啊。总之就是，当时虽然花了很多时间在这上面，但是看着不断增加的 AC 题目，和老师的夸奖，让我感觉很值得。后面期中考试之类的事情，让我感觉自己花的这些时间是有意义的，挺开心的。

由于把 C 语言书配套的 OJ 的题目写得差不多了，我开始尝试 acm 选手经常提到的 cf。

我在 Codeforces 的第一次 VP 和提交，我记得当时是在图书馆打的，虽然一题没写出来，刚开始看英文题真的好不习惯！尽管我高中英语学得还可以。应该也是我第一次去图书馆，是在图书馆的一楼，当时感觉去图书馆那个地方的人挺少的，现在好像多了。今天 11 月 10 日，距离那天，差不多正好两年了。

TIP

VP 指的是 Codeforces 中的 Virtual participation 功能，这个功能可以让用户模拟比赛的场景，计算 AC 与罚时，并且无法查看题解与其他人的代码。是广大 acmer 训练经常用到的功能。其他比赛里的类似功能，也都称为 VP。

重温一下那个题目吧，总之后来看了题解很简单，直接输出字符串的最后一个字符就可以。

题目原文 ​

Let's consider a game in which two players, A and B, participate. This game is characterized by two positive integers, X and Y.

The game consists of sets, and each set consists of plays. In each play, exactly one of the players, either A or B, wins. A set ends exactly when one of the players reaches X wins in the plays of that set. This player is declared the winner of the set. The players play sets until one of them reaches Y wins in the sets. After that, the game ends, and this player is declared the winner of the entire game.

You have just watched a game but didn't notice who was declared the winner. You remember that during the game, n plays were played, and you know which player won each play. However, you do not know the values of X and Y. Based on the available information, determine who won the entire game — A or B. If there is not enough information to determine the winner, you should also report it.

解释 ​

大概意思是说有两个人玩一个游戏，游戏与两个数字 X 和 Y 相关。两个人会一直玩，知道有一个人赢了 Y 局，每一局游戏由多场组成，在每局游戏里先赢 X 局的人获得了胜利。

现在给我们一个字符串，告诉我们每小场游戏是谁赢，我们需要输出最后的赢家。

当时我的代码是这样的，

cpp
#include<stdio.h>
int main()
{
	int n, cnt = 0,plays,i,j,cnt_A=0,cnt_B=0;
	char str[10000],wins[10000];
	scanf("%d",&n);
	for (i = 1; i <= n; i++) {
		scanf("%d",&plays);
		scanf("%s", wins);
		for (j = 0; j < plays; j++) {
			if (wins[j] == 'A')
				cnt_A++;
			else
				cnt_B++;
		}
		if (cnt_A > cnt_B)
			str[i] = 'A';
		else if (cnt_A == cnt_B)
			str[i] = '?';
		else
			str[i] = 'B';
		cnt_A = 0;
		cnt_B = 0;
	}
	putchar('\n');
	for (i = 1; i <= n; i++) {
		printf("%c\n", str[i]);
	}
	return 0;
}

这当时只想着看是 A 多还是 B 多，如果相等就无法判断，哈哈哈，有点想笑。

然后当时我看不懂这种多组测试的，我以为必须读完全部的样例才能输出，后来根据我的经验，大多数人第一次都会这样想，哈哈哈。后面看了题解，说是肯定是最后一局游戏谁赢，那么谁就是赢家，哈哈哈，确实，而且这很好想。

总之就是当时看题也很难受吧，最开始我都不知道有这个翻译插件，好像下一个学期才知道的，读了很久的题，难受死了，虽然没有很难的单词来着，是不是应该比高中阅读简单。

不过这个题目的描述，好像属实也是有点那啥。。。也不能很怪当时的自己吧，现在看这题，你要我翻译讲给别人，可能也很难说清楚。什么 game 由 set 组成，sets 由 play 组成……

这是第二天 AC 的代码

cpp
#include<stdio.h>
#include<string.h>
#include<stdlib.h>
int main()
{
	int n, i,plays,a;
	char str[10000],result[10000];
	char* play[10000];
	scanf("%d",&n);
	for (i = 0; i < n; i++) {
		scanf("%d", &a);
		getchar();
		gets(str);
		play[i] = (char*)malloc(sizeof(char)*(strlen(str) + 1));
		strcpy(play[i], str);
	}
	for (i = 0; i < n; i++) {
		result[i] = play[i][strlen(play[i]) - 1];
		free(play[i]);
	}
	for (i = 0; i < n; i++) {
		printf("%c\n", result[i]);
	}
	return 0;
}

那会看了题解说是直接输出字符串的最后一个，但我当时又一想，好像还真不知道怎么输出字符串末尾。。。呃，有点幽默了，然后翻书复习了一下相关内容。。。反正感觉很幽默，或许我是一直这样，思考很少。

过了几天，发现大多数题解都是 C++ 写的，于是开始用 C++ 的。刚开始我认为 C++ 只是输入输出变成了 cin cout，我知道这样想是不对的，但那让我很积极的去用 C++ 写代码，然后看题解或是别人的代码里不会的东西，就单独学一下。

第四章 迷茫 ​

那场校赛，挺简单的，即使是我这种新生，也写得比较轻松。。。是 5 还是 6 道题，其中一道是英文，我最后一题没写出来，感觉看题好像都看不懂。。。不过后来也没有补题，因为我们学校 OJ 也一直没有开放那些题目来着，不过也无所谓了，是一道 dfs/bfs 的基础题目吧，不过我一次都没有写过这种题目，也不知道这玩意是啥。

最后我居然是第二名，中午的时候有一个神秘人加上了我的 qq 联系我，问我想不想打 acm。后面得知他是大二的 tjf 学长，也是我后来一段时间的队友，他也是被他的老师带动，决定要好好学习算法想参加比赛。

他挺厉害的，也是只学了一两个月，但是就会 dfs 这种高级的内容了，不过他用的是 Java，好吧这更厉害了，因为 C++ 的学习资料总是多一些吧，算法方面。

但是后面其实也没有啥具体的训练计划吧，都是各写各的题目，谭老师给我们拉了个小群，然后建了个共享文档。大家每天把写的题目给贴进去，感觉其实也挺不赖的。也挺可惜当时没有多懂一些东西，比如弄一个团队一起更新的博客，那是不是很酷呢？

后面每次上一些 可以玩手机的 课的时候，我经常会在网络上搜索关于 ICPC 比赛的信息，好像听说由省赛的，打算试一下。但是我从来没有搜到过关于比赛报名的相关信息，我在浏览器和微信公众号文章，都搜索过。感觉很神秘，话说当时为啥没有问一下景老师或者谭老师来着？（……

当时感觉有点难受吧，搜了很多次都没搜到过报名信息。不过前面都是优先准备蓝桥杯吧，学习 C 语言时期，我私下里找过往年蓝桥杯的题目看看，呃，基本一个不会的那种程度，搞得还挺紧张的哈哈哈。

不过之后到寒假结束，基本一直在打 cf 吧，不过别的 oj 没咋用，中间也加了一些算法群，感觉挺有意思的。寒假期间，我 cf 上到了 1200 分，也就是绿名，看到其他群友分都好高，有的写的题好多，感觉很有压力，也惊叹于他人的努力。

第五章 苦修 ​

我们找到了当时校赛第三名的 wls 学长，他是大三的，虽然当时在忙项目之类的，但还是同意与我们一起。

此时此刻，写下这些，我仿佛又回到了那个我们第一次在机房相遇，点开一场湖北省赛题目的那个下午。我们当时看一道通过人数最多的题，都想了非常非常久，讨论了一大堆哈哈哈，纸上画来画去的，还启动 Excel 了哈哈哈。我们当时是在一个自习用的机房，很多学长学姐在里面搞学习，不好意思打扰，后面找了空教室一起过去。

看了题目之后，我们更深刻的意识到了这个比赛的难度，我们的目标从一开始，也一直就是不要垫底，不要给我学校丢脸吧，然后体验比赛氛围。直到练了很多题目之后，心态才发生了一些改变（）。

由于有这个参加比赛的理由，我顺理成章的不用再上晚自习，辅导员爽快的同意了，说只要我是在搞学习就可以。于是我们每个晚上都会一起写题，以及周末。具体细节有点不太记得了，刚开始我们没有专属的教室，一起去空教室 VP，当时的感觉真的好好。教室的环境也挺不错的，但是 VP 的过程真的很坐牢。看着一直超时的题目，感觉很难受，但我们一直在想办法去改，去想其他的思路。

甚至有时候会有点那种神智不清的感觉了，比如，有没有可能 xxx，呃呃，大概这样吧，然后胡言乱语。刚开始 vp 的结果一直很差，即使 vp 的都是省赛，大多也是垫底吧，很难受。

这是当时一起在空教室 VP 2022 湖北省赛时的珍贵影像资料……时间是 3 月 24 日。

这场的 L，我们反复讨论了很久，改了很久代码，比如说加什么小优化啥的，后面一直在想，啊啊啊。然后最后还是显示冰冷的 TLE，我们坐牢了很久很久，这天晚上 wls 学长请我们吃了饭，这是我们第一次一起吃饭！

然后吃完饭回到教室，我们都开始搜这个线段树相关的内容，服了，哈哈哈。一直看不懂，这个啥 build, push up, push down 到底是啥东西，建树是咋建的，总之我们都花了很久，但学习进度就……

坐牢很难受，但我们坐一起吃饭之后，感觉心情好了很多，甚至之后还有点开心。现在想来，还是很怀念。其实那也是，我大学以来，第一次和学校里的同学出去吃饭。虽然我们寝室人都挺好的，关系也都可以，但大家都挺内向的，我也是（），没有人提过类似的事。

跟你们一起吃饭很开心的，就是感觉跟有些朋友一起吃饭，他们不喜欢说话，虽然可能也没啥说的，那样感觉有点无聊。也许是因为跟你们一起玩多了哈哈，我感觉那样啥都不说有点不正常。

那之后，我们经常一起 VP 坐牢，也经常一起出去吃饭，坐牢坐的很多，快乐也挺多，后来还认识了 ✈️ 学长。

由于差不多距离 ICPC 省赛只有一个月，我们都觉得专门学习算法可能帮助不大，不如专心 VP 各种省赛，因此后面 VP 的频率非常高，经常晚上十一二点才回寝室吧。我们寝室楼是十一点关门，晚了回去要叫阿姨起床给我们开hhh，对不起打扰阿姨了。但是没有办法呀，我们没有时间，并且白天要上课，VP 要很多时间。

刚开始是一直都坐牢，但是最后快要去比赛时，我们 VP 很多场是可以得到奖的！于是我们的心态开始发生了转变，VP 的目标开始变成争取进前 60% 。

哈哈哈，当时我们课都挺多的。然后晚上为了 VP 经常互相帮忙带饭，经常就吃完直接启动。真令人怀念啊。

VP 的过程中，我感觉我有点，就是我有时候感觉不想写了，唉，感觉有很多地方挺对不起你们的。如果没有你们，我绝对会更差，假如我们三个，但凡有一个人不那么积极，结果都会不一样的。感谢你们呀！

第六章 小试牛刀 ​

在这样充分的训练中，省赛开始的日子很快到了。出发的前一天晚上，我们一起去散步，大家都在想着比赛的事情吧，谁也不知道结局会是咋样。

很可惜的是，这次比赛场地并不是经典的体育馆，而是在机房。对于来之不易的参赛机会，我们格外珍惜。第一天的热身赛，我们像往常一样三个人看同一道题，结果貌似还行的，也拿到了 ICPC 生涯的第一个气球。

比赛是在 HUST 的网安学院办的，他们一个学院的面积，好像比我们一个学校还大？感觉环境挺不错的，楼也很新。

这天晚上睡得很顺利啊好像，不错……第二天走进机房前，想法有点复杂，感觉我们可以拿到牌子，又感觉拿不到。因为我们最后 VP 的也不是都稳拿，而且很多省情况应该跟我们不太一样。并且我们本来也没啥实力。

刚开始一小时很顺利，写出三道题，排名也还可以。

噢噢，我们是先看了一下 L 题，讨论了一下感觉很简单啊，直接写然后果然 WA 了，当时很多厉害的队伍 L 题都 WA 了非常多次，于是我们想着先放一放去看看通过人数更多的 J 题。

但是后面，卡了很久，看傻了哈哈哈。这个 J 题当时人均都过了，好像通过人数比 B 还多。但我们就是看不明白，分数怎么能取模呢……为啥他们都会写啊……

然后我们自闭了很久，后面 wls 提出要不我们猜一下，反正也这样了。弄了一下发现正好有样例能对，但是有的不对。瞎搞了一下样例过了就交，搞了很多发。

我去旁边翻了一下算法书，试图找到能有帮助的内容……好像也翻了一会，有一行小字提到了分数取模，要用费马小定理。知道怎么取模之后，我们又一起瞎搞了很多，最后成功命中了！

这个 L 居然最后一小时搞出来了，这个题最后正式队 1200 发提交，只过了 86 队，很夸张。我们刚开始交的，只考虑了一点点情况。这个题就是分类讨论，其实没啥算法，但是情况挺多的，后面我们每次交基本都是加了新的 case。但是开局就满足条件的，我感觉不能确定 cost 是不是 0，于是我们每次提交交两次，一次特判一次不特判……

就一直在那搞，感觉头都昏了。。。最后已经不抱希望了，因为没时间了，并且加了那么多次东西，结果永远是红色。队友改了一下，还是 WA，结果把特判加上之后，居然就过了。

之后就还有十分钟才结束，现在已经忘记当时聊了些什么，我们当时拍了桌子吗？不记得了，不过肯定当时非常开心。

乐，狂砍 1145 罚时。尽管只是省赛，尽管只是铜，但那也是刚开始我们自己都没有想过的结果，可能也基本没有人觉得我们会拿到奖。

感觉很极限了，但凡有一个人没有尽力，都不会这样。

我们一起经过了一次又一次坐牢，每天晚上都是寝室关门了才回去，唉唉，其实离开你们之后，我好像再也没有经历过那样的生活了。虽然后面其实也写了很多题，但没有当初的那种强度了。

感谢一直默默支持鼓励我们的老师们，也一直为我们争取机会。

我们只是这所学校的过客，但也留下了一些痕迹。

盒一下自己，应该没事吧（）。

第七章 新的战斗 ​

省赛和五一基本是挨着的，我们三人五一假期一起去了长沙玩，虽然那几天赶上下暴雨，也没去啥地方，而且长沙好多人……这是除了我家和学校以外，我去的第一个其他的城市。

有一天下午我们去了橘子洲走路，走了好久，那里好大。。。也是在那里，我们顺便看了一下，有其他 ICPC / CCPC 邀请赛可以报名。

于是我们又报了昆明邀请赛，感谢老师们帮我们申请比赛机会！谭老师高瞻远瞩，让我们报名昆明而不是其他的站。话说那会昆明群的文档显示的截止报名时间已经过了，反正非常幸运吧。如果五一我们没有一起出去玩，也许不会关注这些了。

于是过完这个假期，我们又开始继续准备战斗了。

过程其实和准备省赛差不多，我们又一起 VP 了一堆比赛，刚开始也是咋都没有牌，后面也差不多能有。回寝室的时间也基本没有变。

我记得就是那会有时候回寝室，室友有时候都关灯睡觉了。那会都比较健康，现在都睡得更晚了……

昆明好远，我们六点多起来出门，晚上九点才到昆明，坐的是高铁，也这么慢。而且路上很多时候网络都不太好，挺难受的，不过这次谭老师陪我们一起坐车，稍微好点。

这一章先简单写一点吧，以后如果想到，可以再补上。

正式赛那个 A 题，很简单，就是个模拟题，但是我就是不知道哪错了。。。这种情况，很麻烦，有的可能就是很难找到。还好 wls 来重新写一下，很快就过了。

我把代码打印出来，带回来了，后面发现，只是一个下标写错了，大概是 idx 和 idx - 1 这种差别了。总之有队友真好。

后面卡了，这个 E 挺难的，完全不知道怎么开始写代码。不过后面我们三个有集中精神一起弄，居然 AC 了。这里咋说呢，当时当然非常开心。后面知道了一些情况，这里就不深入说了。。。

那天晚上，我们四个人一起，在外面逛了好久，当时感觉好累，头晕，但是非常开心！也只有经过那么多的付出，有了幸运的结果，才能有那种心情，真的很不容易。

记得这几天好像吃了很多预置菜来着……哈哈哈。

当我再次回到这座城市，我感觉自己像是小说主角，尽管能力并不强，但是身边所有人的人对我都很好！

篮球杯国赛的时候，cf 才刚上 1400, ICPC 省赛的时候是 1200，菜完了。

忙碌的生活终于结束了，我的大一也要结束了。我篮球杯省赛排在很后面来着，差一点就没省一了，国赛也果然就混了个国三，感觉基本没有完全 AC 的题目，省赛打完之后也是感觉很不爽。我后面就写题没那么多了，但他两还在每天花那么久写项目，唉唉，还是太努力了！

第八章 不存在的 AC ​

之后便是 ICPC 和 CCPC 网络赛，每一场打得都很烂，虽然也是早就料到的结果，而且其实也还好，因为一年之后的我还是差不多的……和 txf，hlj，tjf，wzj 一起打的，感谢你们陪我。

大一暑假，我写的题有点少，而且基本全是水题。本来开始想报多校的，但是不能确定以后谁能陪我一起打，单人报又有点不舍得，找网友拼有点不好意思，于是没报了。其实都是借口罢了，虽然我现在感觉就算报了，可能也，比赛是好的，是我的问题。

当时洛谷绿题和 cf 1600 的题加起来可能才二三十吧，于是我后面专门多练了一些 1600 的题，差不多开学一个月后上了 1600，当时很开心。但这也是我 ICPC 生涯的最后一个段位了。

当时那场的开始时间是 03:35，哈哈哈。

上半年的昆明邀请赛，给每个获奖学校发了一个区域赛名额，如果没有这个，这年应该不会去比赛了。这一场并没有拿到奖牌，但其实我们也练了很多的。我们把近两年的区域赛基本 VP 完了，没有一场能拿牌子的哈哈哈，24 的 CCPC 重庆和 ICPC 沈阳是最接近的，打的体验也是相对最好的。

这一段时期应该也是坐牢最惨的时期……期间 tjf 还因为伤病差点赛季报销了，这期间我们 VP 了很多场，大多数时候都非常坐牢。经常几小时，可能只有最开始一小时才有 AC 吧，后面花很多时间讨论，但没有收获，看了题解基本也是很难看懂的那种……

但是那两场 VP 我们真的非常有希望，而且我们基本都是晚上下课了 VP，基本都是打三四小时。可是去昆明之后，人傻了啊。签到题我们一小时才做出来，第二个签也花了很久，而且基本一小时，才大部分队签上到吧，开始都是人很少，吓哭了。

后面一个计算几何，应该不是很难啊，但我之前从来没用过 atan 这些，当时可能哪里精度有问题。就是一直搞不出来了，就结束了～

跟 tjf 和 wzj 一起去的昆明，其实也还可以啦，虽然没有拿到奖，但也很怀念和你们一起的时光。打铁的滋味，确实很难受！

也是在那个时期，我弄了自己的第一个博客，记了一些东西，虽然后面打多了懒得写了。那个博客的内容，我后面可以弄到这边来。

寒假报了牛客的寒假营，同样烂完了，但是在这个时期，我认识了 @muyuxi，他是和我同一个城市的另一所学校的，跟我同年级。之前很早就期待认识他们学校的学生，希望他们也一起打，因为省赛那时，一直没见过他们学校打比赛，但暑假多校看到了。在一个群里看到的，他 qq id 和牛客 id 是一样的，于是我就加好友了，也许这就是喜欢偷看群友聊天的好处吧。

之后，就是今年上半年的邀请赛，我打了三场……两场都是打铁。

武汉和西安是和 hlj 和 yg 一起去的，前面拉着他们练了一些，但是后面发生了一些事情，我的心态也有点不太好，对不起你们。他们也是我打 ICPC 唯一和我一起自费了打的队友，我真的很抱歉。我自己也是，去第一场邀请赛，武汉之前，我以为肯定能拿奖，赛时打得确实还行，最后一个就是调不出来。比赛结束直接去车站了，晚上 11 点到学校。

后面打西安的时候，在车上和酒店又看了这个题，在知道做法的情况下，还是干了几小时才写对，唉，哈哈哈。武汉那场，去看滚榜了，刚坐下来开始滚榜，我就出去了。因为正好就是从我们上面一名开始滚！最后差 2 罚时，哈哈哈！

大家都太强了，太强了。西安，一样的，不，不一样，打得烂多了。离奖牌差远了，唉，太弱小了，氛围也很压抑，后面也不好意思再拉他们两打了。

南昌跟 ls 和 txf 一起去的，我们去的前一天晚上，老师和我说可以报销……本来是想好的自费去的，也做好了打铁的准备。我在比赛开始的前十天才拉他们说一起打比赛，也很感谢你们愿意跟我一起去，我们基本每天都 VP 一场。同时，这两位也是当时陪我聊天最多的两个人，挺开心的。

南昌那天晚上，一晚上没睡，一直躺着睡不着啊，然后干脆起来写了两个题，五六点稍微睡了一两小时。很幸运这场拿了奖，txf 真的帮我很多，一直造数据，以及想出了最后一个 AC 的题。再后面一个题，其实很简单，我赛后稍微想了一下，不是怎么这么简单？？？但当时就是，感觉。

为了这场比赛，我们来回坐了 23 小时硬座……而且赛后也发生了一些难过的事情，好累，好难受。

这一次篮球杯倒是还行，拿到国一了，虽然是 B 组，但是体验比去年好多了。

第十章 金盆洗手 ​

今年的暑假，或者更早的时候，我就已经想好了，把今年能打的区域赛打完，就金盆洗手了。

暑假我报名了牛客多校，我一个人报的，咋说呢，感觉打了跟没打一样，题是很好的，但我人不行。多校的题，基本全都没补。感觉不如不打的，暑假去赚点报名费，这样说不定可以自费打一场（）。

三场网络赛也都是一个人打的，ICPC 第一场运气不错，后面的两场网络赛，其实我没打……ICPC第二场网络赛，一题没写出来，虽然几个题基本全都过了，但我真的，看了三个小时，那几个基本都看了。我根本不知道怎么想。然后三小时就提前结束了，没有交过一发提交。

CCPC 跟没打没区别，跟去年差不多的排名，，，哈哈哈，这是在干啥？我也不知道，但是我写不出来的题，感觉就是写不出来。他们有的人可以多想一段时间能写出来，我觉得我一直是只能写出那种，很快有思路的题。

练得太少了吧，思考不够。但是很幸运的是，ICPC 沈阳，校排前 220 给一个名额，我们网络赛校排正好在 [210,220] 之间，拿到了一个名额，不赖。

因为一些原因，没法去沈阳了，很幸运申请到了武汉的外卡，于是这就是我的最后一次了。跟 sxb 和 zjq 一起去的，他们也很好，感觉也是有点对不起他们，但是早就知道不可能拿到奖牌，最后并没有那么难受。事实确实如此，最坐牢的一集。

如果我可以在比赛开始前存档的话，感觉这次读档 100 次都拿不到奖牌，就是这样的。但是也不是 bad ending，我认识了许多伙伴，可以说，除了奖牌，我啥都有了。也可以是 good ending 吧。

相比大多数选手，我挺不努力的，但是比起以前的自己，我感觉进步很多。在 XCPC 的剧情中，我收获了许多。可惜大学只有一次，也只有一条线，如果可以，我也想看看其他线的自己。
